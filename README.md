# Aplikasi Login Firebase

## Deskripsi
Aplikasi mobile Android yang dibangun untuk memenuhi tugas Modul Bab 3–5.
Aplikasi memiliki fitur login menggunakan email dan password menggunakan Firebase Authentication.
Aplikasi juga memiliki fitur input dan hapus data yang terintegrasi dengan Firestore dari Firebase.

## Fitur
- Login Email & Password
- Firebase Authentication
- Input & Delete Data
- Firebase Firestore
- Android Release Build
